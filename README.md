프로젝트 개요

이 프로젝트는 단순한 포트폴리오 사이트가 아닙니다.

최남호라는 사람의 경험, 가치관, 프로젝트, 결과물을 소개하는 개인 브랜드 사이트입니다.

방문자가 단순히 경력을 읽는 것이 아니라,
스크롤을 내리면서 하나의 스토리를 경험하는 홈페이지를 목표로 합니다.

최종 목표

이 사이트는 단순한 포트폴리오가 아닙니다.

“최남호”라는 사람을 소개하는 개인 브랜드 홈페이지입니다.

방문자가 1~2분 정도 둘러본 후

“이 사람은 교육행정 경험도 있고,
IT도 직접 해봤으며,
실제로 결과물을 만들어낸 사람이구나.”

라는 인상을 받을 수 있는 사이트를 만드는 것이 목표입니다.
